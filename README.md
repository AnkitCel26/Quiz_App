# Quiz_App
A simple and interactive Quiz_App using HTML,CSS,JAVASCRIPT.
This app fetch the questions dynamically using API so that user get different types of question on every new attempt.

Tech Stack:-  
1.HTML  
2.CSS  
3.JavaScript  

Features:-   
RealTime Question  
Instant answer selection  
Final score display

API used:-  
https://opentdb.com/api.php?amount=10&category=9&difficulty=easy&type=multiple

How it works:-  
1.Fetches quiz questions from API  
2.Displays one question at a time  
3.User selects an answer  
4.Score updates based on correct answers  
5.Final result shown at the end
